                          ▸ src/plan_tasks.mjs::runDailySchedules

                          affection 默认 35/100 而非 0；stage 默认 '暧昧'
                          可演进：暧昧→恋人→深爱（也可手动回退到朋友/陌生人）
                          每阶段称呼 / 撒娇 / 话题深度差异化
                          阈值：0-14陌生 / 15-29朋友 / 30-54暧昧 / 55-79恋人 / 80+深爱
                          ▸ src/memory.mjs::computeRelationshipStage
                          ▸ src/companion.mjs 暧昧/陌生人/朋友档都加"她心里
                            悄悄喜欢他"的人格指令（v1.4.2）

重置为暗恋初心 (v1.4.2)   Dashboard 一键按钮把 affection/stage/mood/dependency
                          拨回默认起点，聊天/记忆/日记保留
                          ▸ POST /api/companions/:id/reset-to-crush

像真人发微信              ≤15 字一条、多条 || 连发、剥离 AI 味
                          ▸ src/bot.mjs::splitReplySegments

人设守卫 Persona Guard    回复后一致性校验，自动检测"我是 AI"、客服话术、
                          阶段违规；轻问题后处理，重问题重生成
                          ▸ src/persona_guard.mjs

【睡眠 / 作息系统】src/sleep.mjs + companion_sleep_schedule / companion_missed_messages
  · 默认 00:30 睡 / 07:30 起，每天不对称抖动（睡 -15/+45、起 ±10 分钟）
  · 睡眠时段 微信(bot.mjs) + 网页(playground.mjs) 都静默拦截，消息入 missed 队列
  · 睡前发"我睡了"晚安 → 挽留窗口（用户"再陪陪我"延后 20min）→ sleep tick 在 bed_at 真入睡
  · 起床 proactive morning：早安 + "昨晚睡着了"未读摘要补回
  · 📞 打电话叫醒她（dashboard，脉冲发光）：立刻 exitSleep + 起床气，AI 发被吵醒短消息
  · dashboard iOS 风格圆形拨盘（拖 🌙☀️ 双把手 + 睡眠时长，松手自动保存）
  · 前 7 天观察用户作息 → 第 8 天中位数固化（可手动设/关闭）

【安全 / 防刷】Cloudflare Turnstile
  · 注册页(auth.html) + 找回密码页(forgot.html) 发邮箱验证码前先人机验证
  · 后端 send-code 先 verifyTurnstile(token, remoteIp) → 官方 siteverify
  · secret 仅 .env(TURNSTILE_SECRET)，site key 前端硬编码；未配置则跳过（dev 友好）
